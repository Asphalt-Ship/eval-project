A propos de ce site

Ce site est un projet personnel regroupant l'ensemble des techniques et technologies relatives à ma formation 'développeur web / web mobile' qui ont été abordés lors de mon stage en entreprise.

Le design de ce projet s'inspire du site web que j'ai réalisé pour ladite entreprise mais ne s'en approprie aucunement les ressources. Tout élément présent dans ce site a été créé par mes soins.

Ce site utilise des médias libres de droits uniquement.
